# cpp-db-sqlite
Example of DB usage with C++ and sqlite

## Env

Built using VS2019, on Win10

<b>Compiler:</b> msvc

<b>DB lib:</b> sqlite3

<b>Language:</b> C++

## Features

- Creates a .db file, if not exists
- Creates a table
- Inserts into a table
- Selects rows from a table
- Functions as a very simple helper for DDL / DML
